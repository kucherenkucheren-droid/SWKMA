# Правила проекта SWKMA

## Проект
SolidWorks аддон для автоматизации рутины инженера-конструктора.
Аналог SWPlus — поэтапный перенос функционала с нуля.
Первый модуль: спецификация по ГОСТ 2.106-96 (ЕСКД).

## Стек
- C# + .NET Framework 4.8.1
- WPF (интерфейс)
- SOLIDWORKS API
- SolidWorks 2026

## Инструменты
| Задача | Инструмент |
|--------|------------|
| Сложная архитектура, трудные ошибки | Claude |
| Мелкие правки, повторяющийся код | Codex CLI |
| Картинки, визуализация | Gemini |
| Прототип UI | Claude Design |

## Модели — указывать ПЕРЕД каждой задачей
**Claude:**
- Sonnet — стандартные задачи
- Opus — сложная архитектура, важные решения

**Codex:**
- gpt-5.1-codex-mini — мелкие правки
- gpt-5.3-codex — стандартные задачи
- gpt-5.4 — сложные задачи

## Порядок разработки
1. Сначала UI макет в Claude Design
2. Потом реализация в WPF
3. Потом логика (SOLIDWORKS API)
